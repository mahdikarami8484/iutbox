# IUTBox Documentation

## Setup Guide
1. Fork the repository to your GitHub account.
2. Create a new app password for IUTBox.
3. Configure GitHub Secrets:
   - `IUTBOX_USER`: your username
   - `IUTBOX_PASS`: your app password

## Usage Instructions
- After setting up, you can use the direct download feature. 

## Troubleshooting
If you face any issues, check the configuration of your secrets and ensure they are correctly set.
